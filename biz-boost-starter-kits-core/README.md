# Biz Boost Starter Kits Core

The core package for the Biz Boost Starter Kits project, containing shared functionality and components.

## Features

- Core business logic
- Shared utilities
- Common components
- Type definitions
- Helper functions

## Installation

```bash
npm install @biz-boost/starter-kits-core
```

## Usage

```typescript
import { Component } from '@biz-boost/starter-kits-core';

// Use the component
<Component />
```

## Development

### Prerequisites
- Node.js 18 or higher
- npm or yarn

### Setup
1. Clone the repository
2. Install dependencies:
```bash
npm install
```

### Building
```bash
npm run build
```

### Testing
```bash
npm test
```
